# Install  [Node](https://nodejs.org/en)

```ts
node -v 
```
Verify if the node was successfully installed

# Install [Yarn](https://yarnpkg.com/)
```ts
npm install yarn -g
```
# Dependencies of installation projects
Execute commands in the project root directory 
```ts
yarn install
```
# Startup Project
```ts
yarn dev
```
# Pack Project
```ts
yarn build
```
